Materi yang ditawarkan

PostgreSQL

PostgreSQL adalah database yang banyak digunakan pada web app, aplikasi mobile, dan aplikasi analytics.

Python

Python merupakan salah satu bahasa populer yang berkaitan dengan Data Science, Machine Learning, dan Internet of Things (IoT).

Tableau

Tableau merupakan alat visualisasi data yang kuat dan paling cepat berkembang yang digunakan dalam Industri Intelijen Bisnis.

HTML

HTML adalah kependekan dari Hypertext Markup Language. HTML merupakan bahasa yang digunakan untuk membuat halaman web.

CSS

CSS adalah singkatan dari Cascading Style Sheet. Markup language ini biasanya dipasangkan dengan markup language (bahasa marka) seperti HTML.

JavaScript

JavaScript atau singkat dari JS merupakan bahasa pemrograman jenis interpreter, sehingga tidak memerlukan compiler untuk menjalankannya. JavaScript memiliki fitur-fitur seperti berorientasi objek, client-side, high-level programming, dan loosely typed.

PHP

PHP (PHP: Hypertext Preprocessor) adalah sebuah bahasa pemrograman server side scripting yang bersifat open source.

Node.JS

Node.js adalah runtime environment untuk JavaScript yang bersifat open-source dan cross-platform. Dengan Node.js kita dapat menjalankan kode JavaScript di mana pun, tidak hanya terbatas pada lingkungan browser.
